# Branch Digital Collaboration

This repository contains documentation, templates, and planning notes for running secure, organised digital collaboration for an activist union branch.

Focus areas:
- Microsoft Teams structure (channels, roles, permissions)
- SharePoint/OneDrive document organisation and versioning
- Meeting documentation templates and decision logs
- Security-aware working practices for non-production testing

## Scope
This is a documentation and workflow repository. No personal data is stored here.
